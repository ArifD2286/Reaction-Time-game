# Reaction Time game

Simple reaction time game where you need to react as quickly as possible to the signals, which is the color change on the screen and a beep sound.
React as quickly as possible and get the best time possible!!

Made for OneKey by HackClub

### How to play:
- Press [SPACE] to start or restart the game
- React as quickly as possible by pressing [SPACE] when the signal comes
- Signal is change of color on screen and a beep sound

### Made using:
- Godot and coded in gdscript language (main language of godot)
